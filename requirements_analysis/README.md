# Requirements Analysis

This folder contains all preliminary research and requirements analysis for the Hypernoa ADHD app project.

## Folder Structure:
- `adhd_research/` - ADHD-specific research, symptoms, challenges, and user needs
- `functional_req/` - Functional requirements and features
- `market_analysis/` - Competitor analysis, market research, existing solutions
- `non_functional_req/` - Non-functional requirements (performance, security, UX)
- `user_stories/` - User personas, scenarios, and use cases

## Purpose:
To thoroughly understand the ADHD space, user needs, and market opportunities before starting development.
